Guild Roster Tool

This tool will allow you to synchronize the data in the database with a guild dump. Please excerise extreme caution when using this tool as it WILL perform mass updates to the database with no immediate way to restore the data. Only character data such as Level, Class, Race, Rank will be updated by this operation. This will NOT create new characters nor delete characters.
